## Running a Notebook

**Local**

```bash
git clone https://github.com/tomskija/Jupyter-Notebook-Workflows.git
cd Jupyter-Notebook-Workflows
pip install -r requirements.txt
jupyter lab
```

Open any notebook from the `notebooks/` directory.
